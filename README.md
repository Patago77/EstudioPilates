
# 🧘 Frontend – Estudio de Pilates

Este es el frontend de una aplicación web creada para un estudio de Pilates. Permite registrar pagos de alumnos, visualizar información financiera y ofrecer una experiencia clara, organizada y adaptable a todo tipo de pantalla.

## 🌐 Funcionalidades

- Registro de pagos con formulario validado
- Tabla de historial con botones de edición y eliminación
- Visualización de ingresos mensuales y pagos vencidos
- Dashboard informativo y adaptado a celulares
- Estilo moderno con Bootstrap y alertas interactivas

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (vanilla)
- Bootstrap 5
- SweetAlert

## 🖼️ Diseño Responsive

La interfaz se adapta a móviles, tablets y pantallas grandes para facilitar la gestión desde cualquier lugar.
